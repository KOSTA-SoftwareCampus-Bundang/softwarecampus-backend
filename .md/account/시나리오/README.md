# 테스트 시나리오

이 폴더는 기능별 테스트 시나리오를 관리합니다.

## 📂 시나리오 목록

### 회원 관리 (Account)
- [회원가입 시나리오](./signup_scenarios.md) - Phase 5, 6, 7
- [프로필 조회 시나리오](./profile_scenarios.md) - Phase 5, 6, 7
- [로그인 시나리오](./login_scenarios.md) - Phase 16 (예정)

---

## 📖 사용 방법

### 1. Phase 작업 시
각 Phase 설계 문서에서 해당 시나리오 문서를 참조합니다.

**예시:**
```markdown
## 테스트 방법
- [회원가입 시나리오](../../시나리오/signup_scenarios.md) 참조
```

### 2. 테스트 작성 시
시나리오 문서의 Phase별 섹션을 참고하여 테스트 케이스를 작성합니다.

- **Phase 5 (Service Layer)**: Service 메서드 동작 시나리오
- **Phase 6 (단위 테스트)**: Mockito 기반 테스트 케이스
- **Phase 7 (API 테스트)**: HTTP 요청/응답 시나리오

### 3. 시나리오 업데이트
요구사항 변경 시 해당 시나리오 파일만 수정하면 모든 Phase에 자동 반영됩니다.

---

## 🎯 시나리오 작성 원칙

1. **Given-When-Then** 형식 사용
2. **Phase별 섹션** 분리 (Phase 5, 6, 7 등)
3. **정상/예외 케이스** 모두 포함
4. **예상 결과** 명확히 기술
