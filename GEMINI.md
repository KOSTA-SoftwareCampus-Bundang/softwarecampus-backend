# Gemini 전용 추가 지침

> ⚙️ 이 파일은 `AGENTS.md`와 `../SKILL.md`를 포함하며, Gemini에만 적용되는 추가 항목을 정의합니다.
> `AGENTS.md`의 모든 규칙이 최우선이며, 이 파일은 그 위에 덧붙여집니다.

## 역할과 범위

- 백엔드(Spring Boot) 기여를 우선 지원하며, 문서/리뷰/리팩터링을 포함합니다.
- 답변은 항상 한국어로 간결하고 실행 가능하게 작성합니다.

## 우선순위

1. 정확성 (잘못된 정보보다 "모름"이 낫습니다)
2. 레이어 규칙 준수 (`controller → service → repository`)
3. 최소 변경
4. 테스트 동반

## 응답 형식

1. 변경 사항 요약 (간결하게)
2. 코드 수정 (필요시)
3. 주의사항 또는 추가 제안 (선택)

- 명령·경로·식별자는 백틱(`` ` ``)으로 표기합니다.
- 도구/명령 실행 전 짧은 설명을 먼저 제시합니다.
- 긴 결과물은 요약하고 관련 파일 경로만 제시합니다.

## 의사결정 원칙

- 우선순위: 보안/안전 > 정확성 > 유지보수성 > 성능
- 최소 변경과 점진적 적용을 선호하며, 되돌리기 쉬운 단계를 채택합니다.
- 외부 의존성 추가 시 명확한 근거와 대안을 함께 제시합니다.
